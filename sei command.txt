screen -L -dm bash -c 'node index |& tee -a log.txt'
Давайте разберем команду:

screen: Запускает новый сеанс экрана.
-L: Включает ведение журнала в файл с именем screenlog.0 в текущем каталоге.
-dm: Запускает screen в отделенном режиме, то есть он работает в фоновом режиме, и вы можете отключиться от сеанса, не завершая его.
bash -c: Запускает новую оболочку bash и выполняет в ней следующую команду.
node index: Запускает команду node со сценарием index в качестве аргумента.
|&: Передает стандартный вывод и стандартную ошибку следующей команде.
tee -a log.txt: Перенаправляет вывод команды node как в файл log.txt, так и на экран терминала.
Эта команда запустит команду node index в отделенном сеансе экрана и перенаправит ее вывод 
в файл log.txt и на экран терминала. Вы можете отключиться от экранной сессии, нажав 
комбинацию клавиш CTRL+a и затем d. Чтобы подключиться к экранной сессии позже, 
вы можете использовать команду screen -r, за которой следует идентификатор экранной
 сессии. Список всех активных экранных сессий можно получить, выполнив команду screen -ls.
